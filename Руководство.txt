Чтобы развернуть проект на локальном сервере в стадии "разработка"
необходимо выполнить следующие действия:
1. Создать папку
2. Скопировать в эту папку файлы и папки из репозитория
3. Открыть данную папку в среде разработки
4. Установить node_modules с помощью комманды $npm install
5. Запустить проект с помощью комманды $npm run start ($npm start) 


Чтобы развернуть прект в стадии "production" 
необходимо выполнить следующие действия:
1. Выполнить пункты 1-4 предыдущего руководства
2. Запустить комманду $npm run build 
3. Перейсти в папку build с помощью комманды $cd build
4. Запустить проект с помощью комманды $npm run start ($npm start)
